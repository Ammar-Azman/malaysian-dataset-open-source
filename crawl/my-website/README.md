# Website snapshot

## how-to

1. Put necessary urls in [list.txt](list.txt).
2. Run [run.py](run.py),

```bash
python3 run.py
```

This script is to get all nested href.

3. Run [run.sh](run.sh),

```bash
bash run.sh
```

This script is to fetch full page for each href.
